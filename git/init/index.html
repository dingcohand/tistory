Git Bash 설정

이번 시간에는 Git 설치 후 초기 설정 방법을 알려드릴게요!
windows에서 Visual Studio Code 사용 시 터미널 기본 설정이 `powershell`입니다.
이는 Git을 사용하기에 어려움이 있어, 기본 터미널을 `Git bash`로 바꾸고 시작합니다.

Git을 설치할 때 Git bash도 설치됩니다.
`Command Palette(F1)` → `Terminal: Select Default Profile 검색` → `Git Bash 클릭`
이렇게 하면 이제 기본 터미널이 `Git Bash`로 설정됩니다.
(`+`를 눌러 새 터미널을 열은 후에 맨 처음 powershell은 닫아주세요!)

전역 사용자 설정하기

아래 명령어로 이름과 이메일을 설정할 수 있습니다.

        git config --global user.name "본인 이름"
        git config --global user.email "본인 이메일"

제대로 설정됐는지 확인하려면 아래의 명령어로 확인 가능합니다.

        git config --global user.name
        git config --global user.email

기본 브랜치명 변경하기

branch 이름 기본값이 master로 되어있는데요, 이 용어는 slave 노예를 연상시키는 단어라고 하여
이를 개선하기 위해 브랜치명을 main으로 사용하는 것이 좋다는 의견이 제시되었다고 합니다.
아래의 명령어를 이용하여 기본 브랜치 이름을 main으로 설정할 수 있습니다. 

        git config --global init.defaultBranch main

줄바꿈 에러 방지하기

맥, 윈도우 협업 시 줄바꿈 에러를 방지하기 위해서는 아래 명령어를 사용하면 됩니다. 

      윈도우
      git config --global core.autocrlf true
      맥
      git config --global core.autocrlf input
