Live Server

웹 페이지를 만들 때, 새로고침 하지 않아도 저장할 때마다 반영되어, 변경사항을 바로 확인할 수 있습니다.
꼭!!! 저장해야 변경사항이 반영됩니다.
우측 하단의 `Go Live`를 누르거나, 단축키 `Alt + L` + `Alt + O`를 눌러 실행할 수 있습니다.


Prettier

코드 포맷팅 익스텐션으로,
루트에 .prettierrc 파일을 생성하여 옵션을 자세히 설정할 수 있습니다.
다음과 같이 설정해야 저장 시 코드 포맷팅이 됩니다.
`Settings에 들어갑니다. (Ctrl + ,)` → `default formatter 검색합니다.` → `Editor: Default Formatter를 Prettier로 설정합니다.`

`Settings에 들어갑니다. (Ctrl + ,)` → `format on save 검색합니다.` → `Editor: Format On Save를 체크합니다.`


Auto Close Tag

시작 태그만 작성해도, 닫힘 태그가 자동으로 삽입됩니다.

Auto Rename Tag

태그명을 변경하면 쌍을 이루는 같은 태그의 이름이 자동으로 변경됩니다. (여는 태그 변경 시, 닫힘 태그도 변경됩니다. 혹은 닫힘 태그 변경 시 여는 태그도 변경됩니다.)

CSS Peek

class를 `ctrl + 클릭`하면 해당 class를 사용한 CSS 파일로 이동합니다.

만약 `ctrl + 클릭`이 먹히지 않는다면, 익스텐션을 비활성화(Disable) → 활성화(Enable) → 커맨드 팔레트(F1) → `reload 검색` → `Developer: Reload Window` 클릭하면 됩니다.

HTML CSS Support

HTML 요소의 class 작성 시 CSS 선택자 자동완성 기능을 지원해주는 익스텐션입니다.
CSS 프레임워크를 사용할 때 매우 유용한 익스텐션입니다.

HTML to CSS autocompletion

바로 위에서 설명한 HTML CSS Support의 반대 버전 개념으로,
CSS 작성 시 HTML에서 사용한 속성 이름을 자동 완성해 주는 익스텐션입니다.

htmltagwrap

원하는 범위를 선택 후 Alt + W를 누르면 태그로 감쌀 수 있습니다.
(맥은 Option + W)

Image preview

코드 라인 번호 좌측에 이미지가 표시됩니다.
이미지 url에 마우스를 올려놓으면 자세히 볼 수 있습니다.

Font Awesome Auto-complete & Preview

Font Awesom 사용 시 유용한 익스텐션입니다.
아이콘 클래스명 자동완성 및 미리 보기를 지원합니다.
아이콘 클래스명에 마우스를 올려놓으면 미리 보기가 뜹니다.
만약 미리 보기가 뜨지 않을 때는 `Ctrl + Space`를 누르면 됩니다.

cdnjs

VSC 안에서 cdn을 검색해서 바로 적용할 수 있습니다.
사용방법은 아래와 같습니다.
f1 → cdn 검색 → cdnjs: Search for libraries 클릭 → 원하는 라이브러리 검색

HTML End Tag Labels

닫는 태그에 클래스명을 주석 처리로 표시하여, 어느 역할의 태그인지 구분할 수 있게 해줍니다.
태그가 길고 많을 때 유용한 익스텐션입니다.

HTML Enter++

`Shift + Enter`를 누르면 <br>태그와 줄바꿈이 삽입됩니다.
`Ctrl + Enter`를 누르면 형제 레벨의 <p>태그가 삽입됩니다.

Color Highlight

색상 값의 배경색으로 해당 색상을 보여줍니다.

CSS Initial Value

CSS 속성 기본값을 보여주는 익스텐션입니다.
속성에 마우스를 올려놓으면 기본값을 확인할 수 있습니다.

px to rem & rpx (cssrem)

단위를 단축키로 변경할 수 있는 익스텐션입니다.
px → rem (Alt + Z)
rem → px (Alt + Z)
루트에 `.cssrem` 파일을 만들어 다음과 같이 설정할 수 있습니다. 

      
        {
          "rootFontSize": 10,
          "fixedDigits": 3
        }

`rootFontSize`를 10으로 설정하면 1rem이 10px이라 해당 익스텐션 없이도
빠르게 계산이 가능합니다.
`fixedDigits`는 변환된 값의 소수점 이하 몇 자리까지 유지할지를 결정하는데 사용됩니다.
3으로 설정한 경우 소수점 이하 최대 세 자리까지만 설정됩니다.
